Tentang Aplikasi

Aplikasi Sistem Informasi Cuaca Indonesia memberikan informasi cuaca terkini untuk berbagai kota di Indonesia.

Data cuaca diambil dari OpenWeatherMap API dan diperbarui secara real-time.

Fitur aplikasi:

  • Pencarian kota untuk melihat cuaca
  • Daftar kota populer di Indonesia
  • Informasi cuaca lengkap (suhu, kelembapan, kecepatan angin, dll)
  • Saran aktivitas berdasarkan kondisi cuaca
  • Antarmuka responsif untuk semua perangkat
  • Efek suara dan animasi yang menarik
